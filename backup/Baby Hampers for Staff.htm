Baby Hampers for staff

The employee's manager will notify  SIE-1 Front Desk of the new birth so that an order can be placed for a baby hamper. The following information is requested:

- Name & WWID of the employee

- Name and sex of baby (exact spelling for name is required as the hamper contains some embroidered items)

 - Baby's date of birth

 - Delivery name & address

 - Contact phone number for delivery address


Once the order has been placed, the hamper is usually delivered within 1 week.
